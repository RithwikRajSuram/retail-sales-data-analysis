# Retail Sales Data Analysis using Python

## Project Overview
This project analyzes a retail sales dataset using Python to understand sales amount distribution, sales activity over time, and differences in sales amounts across product categories. The analysis uses exploratory data analysis techniques to identify patterns, outliers, and category-level sales behavior.

This project demonstrates practical data analytics skills including data loading, data cleaning, exploratory data analysis, date-based analysis, and visualization using Python.

## Objective
The main objective of this project is to explore retail transaction data and generate useful business insights from sales patterns.

The analysis focuses on:

- Understanding the distribution of total sales amounts
- Identifying sales patterns across transaction dates
- Comparing sales amounts across product categories
- Detecting high-value transactions and possible outliers
- Creating visualizations that support business decision-making

## Dataset
The dataset contains retail transaction records, including sales amount, transaction date, and product category information.

### Key Columns Used

| Column | Description |
|---|---|
| `Date` | Transaction date |
| `Total Amount` | Total sales amount for each transaction |
| `Product Category` | Category of product sold, such as Beauty, Clothing, or Electronics |

## Conclusion
This retail sales analysis shows that most transactions fall within lower sales amount ranges, while a smaller number of high-value purchases create visible outliers. Sales activity appears throughout the year, and product categories show different transaction value patterns. Beauty and Electronics show wider sales ranges, while Clothing includes noticeable high-value outliers. Overall, this project demonstrates how exploratory data analysis can help understand retail sales behavior, identify category-level trends, and support better business decision-making.
